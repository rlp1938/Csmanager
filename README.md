# csmanager

**csmanager** is used hardlink files into a directory which is synced
to cloud storage.



## SYNOPSIS

**csmanager** \[options] source_dir


## EXAMPLE
**csmanager** --cut-at /xenial *aptsource.lst* *aptdest.lst*
